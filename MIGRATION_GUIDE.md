# 新电脑接力操作步骤

## 1. 克隆仓库

在新电脑选择一个目录，例如：

```powershell
D:\2026
```

然后执行：

```powershell
git clone <你的 GitHub 仓库地址>
```

## 2. 用 Codex 打开项目

让 Codex 打开克隆后的项目根目录，然后先读取：

- `AGENTS.md`
- `PROJECT_STATE.md`
- `Obsidian-Diamond-Image-KB/00-总控/新电脑接力入口.md`

## 3. 用 Obsidian 打开知识库

在 Obsidian 里选择：

```text
Obsidian-Diamond-Image-KB
```

这个库是独立库，不要和其它 Obsidian 库合并。

## 4. 新电脑继续推进时的第一句话

可以直接对 Codex 说：

```text
请先读取 AGENTS.md、PROJECT_STATE.md 和 Obsidian-Diamond-Image-KB/00-总控/新电脑接力入口.md，然后按当前规则干跑 20-50 套提示词，不生成图片。
```

## 5. 不要提交到 GitHub 的内容

- 大量生成图片
- 视频素材
- 压缩包
- 临时缓存
- API Key、账号密码、Cookie

GitHub 只保存规则、知识库、脚本、提示词样本和项目状态。
