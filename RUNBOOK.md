# 本机执行步骤

## 1. 打开项目

当前推荐项目根目录：

```text
D:\2026\钻石自动化生图
```

GitHub 同步仓库目录：

```text
D:\2026\diamond-image-automation-handoff
```

## 2. Codex 先读取

每次继续推进前，让 Codex 先读取：

- `AGENTS.md`
- `PROJECT_STATE.md`
- `Obsidian-Diamond-Image-KB/00-总控/首页.md`

## 3. Obsidian 打开

在 Obsidian 里选择：

```text
Obsidian-Diamond-Image-KB
```

这个库是独立库，不要和其它 Obsidian 库合并。

## 4. 推荐继续推进的第一句话

```text
请先读取 AGENTS.md、PROJECT_STATE.md 和 Obsidian-Diamond-Image-KB/00-总控/首页.md，然后按当前规则干跑 20-50 套提示词，不生成图片。
```

## 5. 不要提交到 GitHub 的内容

- 大量生成图片
- 视频素材
- 压缩包
- 临时缓存
- API Key、账号密码、Cookie

GitHub 只保存规则、知识库、脚本、提示词样本和项目状态。
