# 长期自动化闭环路线图

目标：从钻石电商图片素材开始，逐步做到提示词、图片、视频、账号赛道和发布的自动化闭环。

核心原则：
- 第一阶段先把图片素材关卡做好。
- 图片不合格就不要进入视频环节，避免浪费视频生成成本。
- 每一步都要保存索引、评分、失败原因和可复用经验。
- TikTok 发布先支持草稿和素材包，后续再研究官方 API 或人工审核发布。

阶段 1：批量提示词
- 从本知识库读取硬规则、产品框架、人物框架、场景框架、拍摄框架、低 AI 味规则。
- 先批量生成提示词，不直接生图。
- 对提示词做重复度审核，去掉人物、场景、动作、构图、产品设计过近的组合。
- 合格提示词进入待生图队列。

阶段 2：图片生成
- 每套先生成 product_dna。
- 每套 3 张图必须复用同一个 product_dna。
- 输出全身、半身、产品特写。
- 保存原图、裁切图、提示词、产品 DNA、生成时间和批次索引。

阶段 3：图片自动审核
- AI 审核每张图和每套图。
- 审核维度：产品一致性、钻石真实感、人物真实感、动作自然度、AI 味、视频可用性、商用安全。
- 合格图片进入素材库。
- 可修复图片进入重生队列。
- 严重不合格图片直接废弃。

阶段 4：视频生成准备
- 只使用图片审核合格的素材。
- 按产品品类、人物画像、场景类型、账号赛道分组。
- 为每套图片生成视频脚本、镜头运动、文案方向和发布标签。

阶段 5：TikTok 多账号赛道
- 先做赛道分析和账号分组，不急于全自动发布。
- 20 个账号可按赛道拆分，例如：婚戒、日常轻奢、富婆穿搭、网红随拍、成熟买家、高端定制。
- 每个账号建立素材偏好、内容节奏和禁用风格。

阶段 6：草稿或发布
- 优先支持保存草稿和素材包，方便人工检查。
- 后续再评估 TikTok 官方 Content Posting API。
- 不使用违反平台规则的批量发布方式。

当前优先级：
1. 批量提示词去重。
2. 图片生成质量。
3. 图片自动审核。
4. 合格素材库。
5. 视频和发布闭环。
