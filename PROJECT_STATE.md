# PROJECT STATE

最后更新：2026-06-02

## 项目目标

为钻石电商建立一套长期可放量的自动化图片素材工作流。目标用户主要是美国高消费力女性，素材方向是真实名媛、网红、达人、富婆、成熟买家、高端生活方式随拍，不是硬广棚拍。

最终闭环方向：
- Obsidian 知识库管理规则。
- Codex 批量生成提示词。
- 图片生成。
- AI 自动审核。
- 合格素材进入视频生成。
- 后续扩展到 TikTok 多账号赛道、草稿保存或发布闭环。

## 不要做

- 不要把提示词写成固定剧本。
- 不要固定人物长相。
- 不要每天生成高度重复的人物、场景、动作、镜头。
- 不要让全身图和半身图硬展示产品。
- 不要让一套 3 张图换产品。
- 不要让彩钻超过 10%。
- 不要把品牌名、Logo、同款、复刻写进最终提示词。
- 不要把大量图片直接塞进 GitHub。
- 不要默认保存到 C 盘。
- 不要回到成熟外套贵妇模板。
- 组合键里必须拆分使用 `skin_direction` 和 `hairstyle_direction`。

## 下一步建议

推荐下一步：
1. 根据当前 Obsidian 规则先干跑 20-50 套提示词，检查年龄、无外套、发型、特写动作是否达标。
2. 再生成 2-5 套图片测试，每套 3 张独立图。
3. 至少包含 1 套钻戒。
4. 如果需要测试彩钻，单独指定 1 套彩钻，不要超过比例。
5. 对图片做严格复盘：产品一致性、钻石真实感、人物年轻化、动作自然度、产品展示自然度、AI 味。
6. 根据复盘结果同步回自动审核规则。

- 建立图片审核脚本。
